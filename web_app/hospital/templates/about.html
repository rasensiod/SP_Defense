About Page

This application was created as part of the coursework project for the module ELEC0138 - Security & Privacy at University College London. The application mimics a simple hospital interface for patients and medical professionals to access patient data through a input text box, the contents of which are then used to query a SQL database. Depending on the level of security the user might be able to perform an SQL injection attack or not.
